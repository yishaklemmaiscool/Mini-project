ğŸš€ Features

This project fulfills the following requirements:

Home View: Displays a list of products with Name, Price, and Image.

Add to Cart: Users can add items to their cart. If the item exists, the quantity increases.

Cart View: separate page showing selected items.

Quantity Controls: Increase (+) or Decrease (-) item quantity directly in the cart.

Remove Items: Delete specific items from the shopping list.

Dynamic Total: The total price updates automatically as items are added or quantities change.

Empty State: Shows a friendly message if the cart is empty.
